# 🎬 Salesforce-Movie-Booking-Project

This repository contains the documentation for the **Book Now Movie Booking Application** developed as part of my **Salesforce Internship Project**.

## 📌 Project Overview

The application was built on the Salesforce Lightning Platform to manage:

- Movies
- Theatres
- Shows
- Bookings
- AI-powered movie information using Agentforce

## 🚀 Technologies Used

- Salesforce Lightning Platform
- Apex
- Salesforce Flow
- Agentforce
- Reports & Dashboards
- Validation Rules
- Formula Fields
- Lightning App Builder
